# Ecommerce Store built With React, Redux, Mock API

## About The Project
Features
•	Full stack Ecommerce store built with React & Redux with backend API
•	Features: Multiple language (Arabic, English), User authentication, React router v6.3, Pagination
•	Full shopping cart functionality. Submit order by user.
•	Product search. Sort & Filter products by category, brand, price…
